Hari 45: Mengurutkan Hasil - Klausa ORDER BY di SQL

Kuartal 3: Advanced | Bulan 3: SQL Lanjutan

Selamat pagi, pengatur data SQL! Selama dua hari terakhir, kita sudah fokus pada klausa WHERE dan berbagai operatornya untuk memfilter baris data agar sesuai dengan kriteria yang kita inginkan. Kita sekarang bisa mengambil data yang sangat spesifik.

Namun, saat kamu menjalankan query SELECT ... WHERE ..., dalam urutan apa baris hasil akan ditampilkan? Jawabannya: tidak ada jaminan urutan tertentu! Database mungkin mengembalikan baris dalam urutan penyimpanannya, urutan indeks yang digunakan, atau urutan lain yang paling efisien baginya saat itu. Jika kamu butuh hasil query ditampilkan dalam urutan yang spesifik (misalnya, dari harga termurah, atau berdasarkan abjad nama), kamu perlu memberi tahu database secara eksplisit.

Di sinilah klausa ORDER BY berperan. Klausa ini digunakan untuk mengurutkan baris dalam hasil query berdasarkan nilai dari satu atau lebih kolom.

Apa Itu Klausa ORDER BY?

Klausa ORDER BY ditambahkan setelah klausa FROM dan WHERE (jika ada) dalam pernyataan SELECT.

Struktur dasar pernyataan SELECT dengan ORDER BY adalah:

SELECT kolom1, kolom2, ... FROM nama_tabel [WHERE kondisi] ORDER BY kolom_untuk_urutkan [ASC | DESC];

Mari kita bedah bagian ORDER BY:

  • ORDER BY: Kata kunci yang menandakan dimulainya pengurutan hasil.
  • kolom_untuk_urutkan: Nama kolom yang nilainya akan digunakan sebagai dasar pengurutan.
  • [ASC | DESC] (Opsional): Menentukan arah pengurutan:
    • ASC (Ascending): Mengurutkan dari nilai terkecil ke terbesar (A ke Z untuk teks, angka terkecil ke terbesar, tanggal terlama ke terbaru). Ini adalah urutan default jika kamu tidak menentukannya.
    • DESC (Descending): Mengurutkan dari nilai terbesar ke terkecil (Z ke A untuk teks, angka terbesar ke terkecil, tanggal terbaru ke terlama).

Contoh Penggunaan ORDER BY

Mari kita gunakan tabel Produk sebagai contoh:

1

Tabel: Produk

ID NamaProduk Kategori Harga Stok
P001 Apel Buah 5000 100
P002 Susu UHT 1L Minuman 18000 50
P003 Roti Tawar Roti 12000 75
P004 Pisang Buah 3000 150
P005 Roti Sobek Roti 15000 60
P006 Kopi Instan Minuman 1500 200
P007 Keripik Singkong Makanan Ringan 8000 120
P008 Air Mineral Botol Minuman 3000 300
2

Mengurutkan Berdasarkan Nama Produk (Abjad A-Z)

Kita ingin mengambil semua produk dan mengurutkannya berdasarkan nama secara ascending (A-Z).

SELECT * FROM Produk ORDER BY NamaProduk ASC; -- Atau cukup: ORDER BY NamaProduk;

Hasil: Semua produk akan ditampilkan, diurutkan mulai dari Air Mineral Botol, Apel, Keripik Singkong, Kopi Instan, Pisang, Roti Sobek, Roti Tawar, hingga Susu UHT 1L.

3

Mengurutkan Berdasarkan Harga (Termahal ke Termurah)

Kita ingin melihat produk diurutkan dari harga tertinggi ke terendah.

SELECT NamaProduk, Harga FROM Produk ORDER BY Harga DESC;

Hasil: Produk akan ditampilkan dengan urutan harga menurun: Susu UHT 1L (18000), Roti Sobek (15000), Roti Tawar (12000), Keripik Singkong (8000), Apel (5000), Pisang (3000), Air Mineral Botol (3000), Kopi Instan (1500).

4

Mengurutkan Berdasarkan Stok (Terkecil ke Terbesar) untuk Kategori Tertentu

Kita ingin melihat produk dalam kategori "Minuman", diurutkan berdasarkan stok terendah.

SELECT NamaProduk, Kategori, Stok FROM Produk WHERE Kategori = 'Minuman' ORDER BY Stok ASC;

Hasil: Hanya produk minuman yang akan ditampilkan, diurutkan berdasarkan stok: Susu UHT 1L (50), Kopi Instan (200), Air Mineral Botol (300).

Mengurutkan Berdasarkan Beberapa Kolom

Kamu juga bisa mengurutkan berdasarkan lebih dari satu kolom. SQL akan mengurutkan berdasarkan kolom pertama yang kamu sebutkan, lalu untuk baris-baris yang memiliki nilai sama di kolom pertama, ia akan mengurutkan berdasarkan kolom kedua, dan seterusnya.

ORDER BY kolom1 [ASC|DESC], kolom2 [ASC|DESC], ...
5

Contoh Pengurutan Multi-Kolom

Mengurutkan produk berdasarkan Kategori (A-Z), lalu di dalam setiap kategori, urutkan berdasarkan Harga (termahal ke termurah).

SELECT NamaProduk, Kategori, Harga FROM Produk ORDER BY Kategori ASC, Harga DESC;

Hasil: Produk akan dikelompokkan per kategori (Buah, Makanan Ringan, Minuman, Roti). Di dalam kelompok "Buah", Apel (5000) akan muncul sebelum Pisang (3000) karena diurutkan Harga DESC. Di dalam "Minuman", Susu UHT (18000) akan muncul pertama, diikuti Air Mineral (3000), lalu Kopi Instan (1500). Begitu seterusnya untuk kategori lain.

Apa Selanjutnya?

Kamu sekarang bisa mengontrol urutan hasil query SQL menggunakan ORDER BY. Kamu bisa mengurutkan berdasarkan satu atau beberapa kolom, baik secara ascending maupun descending.

Besok, kita akan belajar cara membatasi jumlah baris yang dikembalikan oleh query menggunakan klausa LIMIT (atau TOP di beberapa sistem database seperti SQL Server). Ini sangat berguna jika kamu hanya ingin melihat beberapa baris teratas atau terbawah setelah mengurutkan, atau jika kamu hanya butuh sampel data.

Kemampuan memfilter (WHERE) dan mengurutkan (ORDER BY) adalah inti dari pengambilan data yang efektif dengan SQL. Teruslah berlatih!

Tips Hari Ini: Gunakan pengurutan multi-kolom saat membutuhkan analisis berjenjang seperti mengurutkan produk berdasarkan kategori kemudian harga.
