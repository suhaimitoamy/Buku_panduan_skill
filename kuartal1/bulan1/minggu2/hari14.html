Hari 14: Review Minggu Kedua & Kuis Santai Lagi!

Kuartal 1: Dasar | Bulan 1: Excel Dasar

Luar biasa! Kamu sudah menyelesaikan minggu kedua dari perjalanan belajar analisis data kita. Dua minggu berturut-turut, konsisten belajar setiap hari – kamu patut bangga pada dirimu sendiri!

Rangkuman Minggu Kedua

  1. Rumus Dasar & Operator: Belajar =, +, -, *, /, ^, %, PEMDAS, dan referensi sel.
  2. Fungsi Dasar: SUM, AVERAGE, COUNT, MIN, MAX, COUNTA.
  3. Sort: Satu/multi kolom, centang "My data has headers".
  4. Filter: Angka, checkbox, teks, tanggal, warna.
  5. Grafik: Bar/Column, Line, Pie Chart via Insert > Charts.

Kuis Minggu Kedua

  1. =10+5*2 hasilnya? → 20
  2. Hitung sel tak kosong? → COUNTA
  3. Sort: centang "My data has headers" agar... → Header tidak ikut terurut
  4. Nilai 75–85? → Between
  5. Nama mulai “Roti”? → Begins With
  6. Grafik penjualan bulanan? → Line / Bar Chart
  7. Proporsi kategori? → Pie Chart
  8. Aktifkan filter di tab? → Data

Selanjutnya

Minggu depan kita belajar IF, VLOOKUP, grafik lanjutan, dan PivotTable. Siap-siap!

Tips Hari Ini: Luangkan waktu untuk review. Ini kunci biar makin paham.
