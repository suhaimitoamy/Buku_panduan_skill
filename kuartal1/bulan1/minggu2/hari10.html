Hari 10: Mengatur Data - Mari Mengurutkan (Sort)!

Kuartal 1: Dasar | Bulan 1: Excel Dasar

Apa kabar di hari kesepuluh ini? Kemarin kita sudah asyik berlatih menggunakan fungsi-fungsi dasar pada data nilai siswa. Hari ini kita akan mempelajari teknik penting untuk mengatur data - mengurutkan (Sort) data di Excel!

Mengurutkan data membantu kita menemukan pola, mengidentifikasi nilai ekstrem, dan membuat data lebih mudah dipahami. Mari kita eksplorasi bersama!

Mengapa Mengurutkan Data Itu Berguna?

🔍

Menemukan Nilai Ekstrem

Cepat melihat nilai tertinggi atau terendah dalam suatu kolom dengan konteks data lengkap.

🗂️

Mengelompokkan Data Serupa

Mengumpulkan item yang sama berdekatan, terutama untuk data kategorikal.

📊

Membuat Daftar Terstruktur

Mengurutkan nama berdasarkan abjad (A-Z atau Z-A) atau tanggal berdasarkan urutan waktu.

👁️

Memudahkan Pencarian Visual

Lebih mudah menemukan nama atau item tertentu jika daftarnya sudah terurut.

PENTING SEBELUM MEMULAI

Selalu pilih seluruh rentang data yang ingin kamu urutkan, termasuk baris judul (header). Jika hanya memilih satu kolom, hanya kolom itu yang berubah urutannya, sementara kolom lain tetap - ini akan merusak integritas data karena data dalam satu baris tidak lagi berhubungan!

Cara Mengurutkan Data di Excel

Fitur Sort biasanya ada di tab Data pada Ribbon. Mari kita praktikkan pada sheet "Nilai Siswa".

Latihan 1

Mengurutkan Berdasarkan Nama Siswa (A-Z)

Pilih Data: Klik sel A1 (No), tahan Shift, lalu klik sel terakhir datamu di pojok kanan bawah.
Buka Dialog Sort: Pergi ke tab Data, di grup Sort & Filter, klik tombol Sort.
Atur Opsi Sort:
  • Pastikan "My data has headers" tercentang
  • Di "Sort by", pilih "Nama Siswa"
  • Di "Sort On", biarkan "Cell Values"
  • Di "Order", pilih "A to Z"
Klik OK dan lihat hasilnya!

Latihan 2

Mengurutkan Berdasarkan Nilai Matematika (Tertinggi ke Terendah)

Pilih Data Lagi: Pastikan seluruh data terpilih.
Buka Dialog Sort: Tab Data > Sort.
Atur Opsi Sort:
  • "My data has headers" tercentang
  • "Sort by": "Nilai Matematika"
  • "Sort On": "Cell Values"
  • "Order": "Largest to Smallest"
Klik OK untuk melihat siswa dengan nilai tertinggi di atas.

Latihan 3

Mengurutkan Berdasarkan Nilai Bahasa (Terendah ke Tertinggi)

Coba lakukan sendiri! Urutkan data berdasarkan Nilai Bahasa dari Smallest to Largest (Terkecil ke Terbesar).

💡
Gunakan langkah-langkah yang sama seperti Latihan 2, tapi pilih "Nilai Bahasa" dan urutan "Smallest to Largest".

Mengurutkan Berdasarkan Beberapa Kolom (Multi-Level Sort)

Bagaimana jika ada nilai yang sama? Multi-level sort memungkinkan kita menentukan kriteria pengurutan sekunder.

Contoh: Urutkan berdasarkan Nilai Sains tertinggi, jika sama urutkan berdasarkan nama (A-Z)

Pilih Data dan buka Dialog Sort.
Level Pertama:
  • "Sort by": Nilai Sains
  • "Sort On": "Cell Values"
  • "Order": Largest to Smallest
Tambah Level Kedua: Klik tombol "Add Level".
Level Kedua:
  • "Then by": Nama Siswa
  • "Sort On": "Cell Values"
  • "Order": A to Z
Klik OK untuk menerapkan pengurutan ganda.

Andi90
Budi90
Cici85

Andi dan Budi diurutkan berdasarkan nama karena nilai Sains sama

Sedikit Pengingat tentang Formatting

Sambil kita mengatur data, jangan lupakan formatting yang sudah kita pelajari. Pastikan tabelmu tetap rapi:

✓
Judul kolom Bold
✓
Gunakan Borders agar jelas
✓
Atur Alignment (nilai rata tengah/kanan)

Apa Selanjutnya?

Besok kita akan mempelajari fitur penting lainnya - Filter Data! Dengan filter, kamu bisa menyembunyikan data yang tidak relevan dan fokus pada data yang ingin kamu analisis. Kombinasi Sort dan Filter akan membuat pengelolaan data menjadi jauh lebih efisien!

Jangan lupa untuk berlatih mengurutkan data dengan berbagai kriteria agar semakin mahir!

Tips Hari Ini: Gunakan tombol Sort Ascending (A-Z) dan Sort Descending (Z-A) di Ribbon untuk pengurutan cepat.
