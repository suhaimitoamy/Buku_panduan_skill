Menggali Lebih Dalam PivotTable: Fungsi & Format

Pelajari cara mengubah fungsi ringkasan dan memformat angka di PivotTable Excel

Selamat pagi, PivotTable explorer! Hari ini kita akan menggali lebih dalam kemampuan PivotTable dengan mempelajari fungsi ringkasan dan pemformatan angka.

Mengubah Fungsi Ringkasan di Area VALUES

Secara default, ketika Anda menyeret field numerik ke area VALUES, Excel menggunakan fungsi SUM. Untuk field teks, Excel menggunakan COUNT. Mari pelajari cara mengubahnya!

SUM (Jumlah)

Menjumlahkan semua nilai dalam kategori

AVERAGE (Rata-rata)

Menghitung nilai rata-rata dalam kategori

COUNT (Jumlah Data)

Menghitung jumlah item dalam kategori

MAX (Maksimum)

Menampilkan nilai tertinggi dalam kategori

MIN (Minimum)

Menampilkan nilai terendah dalam kategori

DISTINCT COUNT

Menghitung nilai unik dalam kategori

1

Latihan 1: Rata-rata Total Penjualan per Kategori

Ubah fungsi ringkasan dari SUM ke AVERAGE

Cara Klik Kanan:

  1. Siapkan PivotTable dengan Kategori di ROWS dan Total Penjualan di VALUES
  2. Klik kanan pada angka di kolom Total Penjualan
  3. Pilih "Summarize Values By"
  4. Pilih "Average"

Hasil:

Kategori Rata-rata Penjualan
Buah Rp 312.500
Minuman Rp 283.333
Roti Rp 366.667
Grand Total Rp 320.000

Cara Value Field Settings:

  1. Di panel PivotTable Fields, klik dropdown di field VALUES
  2. Pilih "Value Field Settings..."
  3. Pilih "Average" di tab Summarize Values By
  4. Ubah Custom Name menjadi "Rata-rata Penjualan"
  5. Klik OK

Tip: Mengubah Nama Kolom

Gunakan "Custom Name" untuk memberi label yang lebih bermakna pada kolom Anda.

2

Latihan 2: Jumlah Jenis Produk Berbeda per Pelanggan

Menggunakan Distinct Count untuk menghitung produk unik

Langkah-langkah:

  1. Bersihkan PivotTable sebelumnya
  2. Seret Pelanggan ke ROWS
  3. Seret Produk ke VALUES (default: Count of Produk)
  4. Klik dropdown Count of Produk di area VALUES
  5. Pilih "Value Field Settings..."
  6. Scroll dan pilih "Distinct Count"
  7. Ubah Custom Name menjadi "Jenis Produk"
  8. Klik OK

Hasil:

Pelanggan Jenis Produk
Budi 2
Citra 3
Dewa 2
Grand Total 7

Perhatian!

Distinct Count hanya tersedia jika Anda mencentang "Add this data to the Data Model" saat membuat PivotTable.

Memformat Angka di PivotTable

Angka penjualan atau rata-rata seringkali lebih mudah dibaca jika diformat sebagai mata uang atau dengan pemisah ribuan.

Mata Uang
Angka
Persentase
Akuntansi

Cara Klik Kanan:

  1. Klik kanan pada angka di kolom VALUES
  2. Pilih "Number Format..."
  3. Pilih "Currency"
  4. Pilih simbol mata uang (Rp/IDR)
  5. Atur jumlah desimal (biasanya 0)
  6. Klik OK

Cara Value Field Settings:

  1. Klik dropdown di field VALUES
  2. Pilih "Value Field Settings..."
  3. Klik "Number Format..." di pojok kiri bawah
  4. Pilih "Currency"
  5. Atur opsi format
  6. Klik OK dua kali

Hasil Format Mata Uang:

Sebelum Format

1250000

Setelah Format

Rp 1.250.000

Cara:

  1. Buka jendela Number Format
  2. Pilih "Number"
  3. Atur jumlah desimal (biasanya 0)
  4. Centang "Use 1000 Separator"
  5. Klik OK

Contoh Format Angka:

Sebelum Format

1250000

Setelah Format

1.250.000

3

Latihan 3: Memformat Total Penjualan sebagai Mata Uang

Terapkan format mata uang pada kolom Total Penjualan

Langkah-langkah:

  1. Siapkan PivotTable dengan Kategori di ROWS dan Total Penjualan di VALUES
  2. Klik kanan pada angka di kolom Total Penjualan
  3. Pilih "Number Format..."
  4. Pilih "Currency" dari kategori
  5. Pilih simbol Rp (IDR) Indonesia
  6. Set desimal places ke 0
  7. Klik OK

Hasil:

Kategori Total Penjualan
Buah Rp 1.250.000
Minuman Rp 850.000
Roti Rp 1.100.000
Grand Total Rp 3.200.000

Sedikit tentang Tampilan (Layout & Style)

PivotTable Styles

Ubah skema warna dan gaya tabel dengan memilih style yang berbeda di tab Design.

Report Layout

Ubah tata letak menjadi tabular untuk tampilan yang lebih mirip tabel biasa.

Grand Totals

Atur tampilan grand total dan subtotal sesuai kebutuhan laporan Anda.

Apa Selanjutnya?

Kamu sekarang bisa mengontrol fungsi ringkasan dan format angka di PivotTable, membuatnya lebih fleksibel dan mudah dibaca.

Besok, kita akan belajar salah satu fitur keren lainnya dari PivotTable: Grouping (Mengelompokkan). Kita bisa mengelompokkan data tanggal (misal, per bulan atau kuartal) atau data numerik (misal, per rentang usia atau harga) secara otomatis.

Teruslah bereksperimen dengan berbagai fungsi dan format yang tersedia!
