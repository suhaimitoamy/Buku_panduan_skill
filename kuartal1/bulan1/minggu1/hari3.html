Hari 3: Mempercantik Tabel Excel & Format Data

Kuartal 1: Dasar | Bulan 1: Excel Dasar

Hai lagi! Hari ini kita bakal bikin tabel belanjaannu makin rapi dan enak dilihat. Kita akan belajar formatting dasar untuk mempercantik tampilan dan membuat data lebih mudah dibaca.

Mengapa Formatting Itu Penting?

Bayangin kamu baca teks panjang tanpa paragraf, tanpa judul, semua huruf sama. Capek, kan? Nah, formatting di Excel fungsinya mirip: biar data lebih jelas, informatif, dan enak dilihat.

Formatting Dasar: Teks dan Sel

Buka file Latihan Excel Hari 2 - Daftar Belanja.xlsx. Gunakan tab Home untuk:

  1. Bold Judul: Blok A1:D1, klik B.
  2. Rata Tengah: Blok A1:D6, klik "Center".
  3. Warna Latar: A1:D1, klik Fill Color.
  4. Borders: A1:D6, klik Borders & pilih "All Borders".

Formatting Angka: Supaya Jelas!

Perhatikan kolom Harga Satuan dan Jumlah.

  1. Mata Uang: Blok D2:D6, klik Accounting Number Format.
  2. Hilangkan Desimal: Blok B2:B6, klik "Decrease Decimal".

Latihan Hari Ini

Coba praktik langsung di file Excel:

  • Bold baris judul
  • Semua data rata tengah
  • Warna latar pada judul
  • Tambahkan border seluruh tabel
  • Format kolom harga jadi mata uang
  • Hilangkan angka desimal di kolom jumlah
  • Bonus: Ganti font atau ukuran judul

Sedikit Penguatan Konsep

  • Formatting tidak mengubah nilai asli, cuma tampilannya aja.
  • Excel bisa nebak tipe data (angka, teks, tanggal) dan ini penting banget buat perhitungan.

Apa Selanjutnya?

Besok kita akan belajar memasukkan data lagi, tapi dengan fokus ke konsistensi dan ketelitian input. Kita juga bakal bikin tabel baru: daftar kontak.

Tips Hari Ini: Format itu bikin data enak dilihat, tapi jangan berlebihan. Tujuan utamanya biar data jelas dan nyaman dibaca.
