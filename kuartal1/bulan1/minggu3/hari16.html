Hari 16: IF Lebih Canggih - Bertingkat, AND, dan OR

Kuartal 1: Dasar | Bulan 1: Excel Dasar

Selamat pagi, detektif data! Kemarin kita sudah berhasil membuat Excel mengambil keputusan sederhana menggunakan fungsi IF. Kamu sudah bisa menentukan status "Lulus"/"Remedial" atau memberikan diskon otomatis berdasarkan satu kondisi. Keren, kan?

Tapi, bagaimana jika kondisinya lebih rumit? Bagaimana jika ada lebih dari dua kemungkinan hasil? Atau bagaimana jika keputusan harus diambil berdasarkan beberapa kondisi sekaligus?

Tenang, Excel punya solusinya! Hari ini kita akan belajar tentang IF Bertingkat (Nested IF) dan cara menggabungkan IF dengan fungsi logika lain seperti AND dan OR.

Ketika Satu IF Tidak Cukup: IF Bertingkat (Nested IF)

Fungsi IF standar hanya bisa menangani dua hasil: value_if_true dan value_if_false. Bagaimana jika kita punya tiga atau lebih kemungkinan hasil? Misalnya, memberi nilai huruf (A, B, C, D) berdasarkan skor.

Di sinilah IF Bertingkat berperan. Idenya adalah meletakkan fungsi IF lain di dalam argumen value_if_false (atau kadang value_if_true) dari IF yang pertama.

Struktur IF Bertingkat (untuk 3 hasil):

=IF(kondisi1; hasil_jika_benar1; IF(kondisi2; hasil_jika_benar2; hasil_jika_salah2))

  • Excel cek kondisi1. Jika TRUE, hasilnya hasil_jika_benar1.
  • Jika kondisi1 FALSE, Excel pindah ke IF kedua.
  • Excel cek kondisi2. Jika TRUE, hasilnya hasil_jika_benar2.
  • Jika kondisi2 FALSE, hasilnya hasil_jika_salah2.

Kamu bisa menumpuk lebih banyak IF lagi, tapi hati-hati! Rumus IF bertingkat bisa cepat menjadi panjang, rumit, dan sulit dibaca/dikelola. Untuk kondisi yang sangat banyak, nanti ada cara yang lebih elegan (seperti fungsi IFS di versi Excel baru atau menggunakan tabel VLOOKUP), tapi penting untuk memahami konsep dasarnya.

Latihan Nested IF: Memberi Nilai Huruf

Mari kita terapkan pada sheet Nilai Siswa. Kita akan memberi nilai huruf berdasarkan nilai rata-rata (misal di kolom G):

  • ≥ 90 : "A"
  • ≥ 80 : "B"
  • ≥ 70 : "C"
  • < 70 : "D" (Remedial)
Nama Matematika Bahasa Sains Sejarah Rata-rata Nilai Huruf Formula
Andi 85 78 90 88 85.25 B =IF(G3>=90,"A",IF(G3>=80,"B",IF(G3>=70,"C","D")))
Budi 92 88 95 93 92.00 A AutoFill
Citra 75 82 68 79 76.00 C AutoFill
Doni 65 72 68 70 68.75 D AutoFill

Langkah Praktik:

  1. Buat kolom baru, misal di I, beri judul Nilai Huruf (I1).
  2. Di sel I3 (baris Andi), tulis rumus Nested IF berikut:

    =IF(G3>=90; "A"; IF(G3>=80; "B"; IF(G3>=70; "C"; "D")))

  3. Bedah rumus (dari kiri ke kanan):
    • Apakah G3 ≥ 90? Jika ya, hasilnya "A".
    • Jika tidak, cek lagi: Apakah G3 ≥ 80? Jika ya, hasilnya "B".
    • Jika tidak juga, cek lagi: Apakah G3 ≥ 70? Jika ya, hasilnya "C".
    • Jika semua kondisi di atas tidak terpenuhi (artinya G3 < 70), hasilnya "D".
    • Perhatikan jumlah tanda kurung tutup di akhir harus sesuai dengan jumlah IF yang dibuka (ada 3 IF, jadi 3 kurung tutup).
  4. Tekan Enter dan gunakan AutoFill untuk menyalin ke bawah.

Sekarang kamu punya klasifikasi nilai huruf otomatis!

Keputusan Berdasarkan Banyak Kondisi: AND dan OR

Bagaimana jika keputusan bergantung pada kombinasi beberapa kondisi?

Fungsi AND

=AND(kondisi1; kondisi2; ...)

Menghasilkan TRUE hanya jika SEMUA kondisi di dalamnya TRUE.

Contoh: =AND(A1>10; B1<5) akan TRUE hanya jika A1 > 10 dan B1 < 5.

Fungsi OR

=OR(kondisi1; kondisi2; ...)

Menghasilkan TRUE jika SALAH SATU (atau lebih) kondisi di dalamnya TRUE.

Contoh: =OR(A1>10; B1<5) akan TRUE jika A1 > 10 atau B1 < 5 (atau keduanya).

Kita bisa memasukkan fungsi AND atau OR ini ke dalam logical_test dari fungsi IF.

Latihan IF dengan AND: Status "Prestasi"

Kita ingin memberi status "Prestasi" kepada siswa yang nilai Matematikanya (kolom C) di atas 85 DAN nilai Sains-nya (kolom E) juga di atas 85.

Nama Matematika Bahasa Sains Sejarah Rata-rata Status Khusus Formula
Andi 85 78 90 88 85.25 - =IF(AND(C3>85,E3>85),"Prestasi","-")
Budi 92 88 95 93 92.00 Prestasi AutoFill
Citra 75 82 68 79 76.00 - AutoFill
Doni 65 72 68 70 68.75 - AutoFill

Langkah Praktik:

  1. Buat kolom baru, misal di J, beri judul Status Khusus (J1).
  2. Di sel J3 (baris Andi), tulis rumus:

    =IF(AND(C3>85; E3>85); "Prestasi"; "-")

  3. Bedah rumus:
    • AND(C3>85; E3>85): Ini tes logikanya. Akan menghasilkan TRUE hanya jika kedua kondisi (C3>85 dan E3>85) terpenuhi.
    • "Prestasi": Value if true (jika kedua nilai di atas 85).
    • "-": Value if false (jika salah satu atau kedua nilai tidak di atas 85). Tanda strip hanya contoh, bisa juga dikosongkan ("").
  4. Tekan Enter dan AutoFill ke bawah.

Latihan IF dengan OR: Status "Perlu Perhatian"

Kita ingin memberi status "Perlu Perhatian" jika nilai Matematika (C) di bawah 70 ATAU nilai Bahasa (D) di bawah 70.

Nama Matematika Bahasa Sains Sejarah Rata-rata Status Khusus Formula
Andi 85 78 90 88 85.25 - =IF(OR(C3<70,D3<70),"Perlu Perhatian","-")
Budi 92 88 95 93 92.00 - AutoFill
Citra 75 82 68 79 76.00 - AutoFill
Doni 65 72 68 70 68.75 Perlu Perhatian AutoFill

Langkah Praktik:

  1. Kita bisa gunakan kolom Status Khusus (J) lagi, atau buat kolom baru.
  2. Di sel J3 (jika menggunakan kolom yang sama, hapus rumus sebelumnya atau timpa), tulis rumus:

    =IF(OR(C3<70; D3<70); "Perlu Perhatian"; "-")

  3. Bedah rumus:
    • OR(C3<70; D3<70): Tes logikanya. Akan menghasilkan TRUE jika salah satu atau kedua kondisi (C3<70 atau D3<70) terpenuhi.
    • "Perlu Perhatian": Value if true.
    • "-": Value if false.
  4. Tekan Enter dan AutoFill ke bawah.

Kombinasi Kekuatan

Kamu bahkan bisa menggabungkan Nested IF, AND, dan OR dalam satu rumus kompleks, tapi sekali lagi, berhati-hatilah agar tidak terlalu rumit. Seringkali lebih baik memecah logika menjadi beberapa kolom bantu.

Apa Selanjutnya?

Wow, logika Excel-mu semakin terasah! Kamu sekarang bisa membuat Excel mengambil keputusan yang lebih kompleks berdasarkan berbagai kondisi.
